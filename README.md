E-Commerce Customer Insights Dashboard

Overview:
Analyzed customer behavior and sales data from an e-commerce dataset to generate actionable business insights using Python and Tableau.

Tools Used:
Python (Pandas, NumPy),
Tableau

Key Analysis:
-Customer segmentation (High, Medium, Low value)
-Monthly sales trends
-Product category performance
-Location-based revenue insights

Key Insights:
-Majority of customers are low-value but contribute highest total revenue
-High-value customers are fewer but highly profitable
-Sales show monthly variation indicating trends
